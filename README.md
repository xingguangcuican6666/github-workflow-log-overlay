# GitHub Workflow Log Overlay

A translucent floating desktop overlay for watching the latest run logs of a GitHub Actions workflow.

## Requirements

- Node.js 20+
- Rust stable
- GitHub CLI (`gh`) in `PATH`
- Linux: WebKitGTK dependencies required by Tauri

The app delegates GitHub access to `gh`. Run `gh auth login` first if the target repository or API rate limits require authentication.

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri -- build --bundles deb,rpm
```

On Windows, use `npm run tauri -- build --bundles nsis`.

Artifacts are generated by the GitHub Actions workflow in `.github/workflows/build.yml`.

## Publish A New Repository

```bash
git init
git add .
git commit -m "Initial GitHub workflow log overlay"
gh repo create OWNER/REPO --public --source=. --remote=origin --push
```
